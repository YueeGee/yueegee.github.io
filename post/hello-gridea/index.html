Gridea——基于Github Page

2020-03-07 2 min read # Gridea

我做出了一个新的白嫖尝试——将博客系统更换为静态博客系统Gridea。

Gridea 是一个静态博客写作客户端，帮助你更容易地构建并管理博客或任何静态站点。

一起使用Gridea。然后，爱上它吧！

特征

📝 使用最酷的 Markdown语法快速创建

🌉 在文章中的任何位置插入图片和文章封面图

🏷️ 标签和分组文章

📋 自定义菜单，甚至创建外部链接菜单

💻 在Windows或MacOS 或Linux上使用此客户端

🌎 使用Github页面或编码页面向世界展示未来将支持更多平台

💬 只需配置和访问Gitalk或DisqusJS评论系统

🇬🇧 使用简体中文，繁体中文，英文

🌁 使用应用程序内的任何默认主题或任何第三方主题，免费主题定制

🖥 使用OneDrive，iCloud，Dropbox等自定义源文件夹并同步多个设备。

当前网站状态

网站托管于：Github Page
CDN加速基于：Cloudflare
图床： GitHub+JsDelivr

最后

网站使用Github Page速度还是有点虚，搭配上CF速度稍稍有点改观。
但也后续可能会进行备案然后转移到Gitee，但这也只是计划。
